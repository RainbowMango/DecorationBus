#装修管家

###简介
装修管家旨在深入到装修过程中的各个环节，减轻装修业主负担。  

只有经历过装修的业主才能明白装修过程中的艰辛、迷茫和幸福。预算、设计、施工、建材选购及后续保修等事项都让业主操碎了心。

装修管家的宗旨是“让装修变得简单”

本IOS应用使用全新的swift语言，一切刚刚开始，欢迎加入~
